Two layers · what each owns
FirewallStack
Generic retry wrapper
Wraps every LLM call. Catches FirewallRejection, decides whether to retry, and bubbles blocks up as a structured LLMResult.
max_retries = 2 — three total attempts per call.
On retry, appends FIREWALL_GUIDANCE to the system prompt (avoid PII, role patterns, exec keywords).
On retry, masks 6+ digit runs in the user message.
Exhausted → returns LLMResult(status="blocked") — callers treat as partial output.
Keeps a step_history of every prompt / message / attempt.
SafeChainAdapter
SafeChain-specific translate + pre-sanitize
Translates platform errors into FirewallRejection and pre-cleans the prompt every call. OpenAI adapter is the dev bypass — no firewall simulation.
403 → FirewallRejection(403) · 400 → FirewallRejection(400).
401 → silent token refresh + one local retry — never bubbles up.
Pre-call: mask 8+ digit runs; replace exec/eval/import/__dunder__ with [FILTERED].
Collapses [system, user, …] into a single human message — SafeChain pipeline requirement.
Neutral role labels: Context: / Request: / Response: instead of [SYSTEM]/[USER].
