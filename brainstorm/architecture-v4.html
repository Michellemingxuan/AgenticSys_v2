Shared Skills
All specialist agents expose the same three prompt-mode skills. Behavior differs via injected pillar instructions.
Report
Generate a full section narrative. Output: Key Findings → Evidence → Risk Implication.
Answer
Answer a targeted sub-question from the orchestrator. Also used in peer review mode to evaluate a peer specialist's output.
Data Request
Declaratively specify required data (tables, columns, filters). Routed to the Repository Agent for validation and gating.
Same Agent, Different YAML Config Per Pillar — Bureau Example
credit_risk.yaml
focus: "Delinquency Risk"
table_scope: ["bureau_full"]
prompt_overlay: "Flag 90D+ Marks"
blocked_fields: ["raw_pii"]
cbo.yaml
focus: "Limit Utilisation"
table_scope: ["bureau_trades"]
prompt_overlay: "Burst Risk Signals"
blocked_fields: ["raw_pii","salary"]
Same Python class. YAML controls focus, table scope, prompt overlay, and blocked fields. Adding a new pillar or changing emphasis = one YAML edit, no code changes.

General-purpose & stateless. Multiple agents handle concurrent specialist requests in parallel. Pillar-awareness lives in YAML, not in the agents.
Shared Skills — Data Agents
All data agents expose the same four skills.
Code Gen
Generate executable Python / SQL from the validated declarative data request.
Execute
Run generated code in a sandboxed environment. Retry on recoverable failures.
Retrieve
Return structured results to the requesting specialist via the Repository Agent.
Analyze
Optional: produce a brief statistical summary or anomaly flag on retrieved data before returning.
Single Control Plane: YAML. Each pillar YAML declares which specialists to call, their analytical focus, table scope, prompt overlays, and the list of blocked data fields. The orchestrator reads both pillar instructions and the reviewer's question — compliance constraints are applied before routing, and pillar instructions are injected directly into each specialist's prompt at call time. No Python changes required to add a pillar or retune a specialist's behavior.
